The purpose of this app will be to allow you to search through movies by genre or director. In addition, you will also be able to mark movies as favorites and save them in your account.

To run the application, navigate to movie_api and enter the command "node server.js"

Endpoints

GET request type

/movies Returns a list of all movies

/movies/:title Returns information about a single movie

/movies/:title/genre Returns the genre and genre description of a movie

/movies/:title/director Returns info of a movie's director

/users/:name Returns a user by name

/favorites Returns a list of movies marked as favorites

/documentation Returns this information

POST request type

/users Registers a new user. Expects a JSON object in the request body. If successful, the JSON object is returned back to the user.

/favorites Adds a movie to list of favorites. Expects a JSON object in the request body. If successful, the JSON object is returned back to the user.

PUT request type

/users/:name/username Updates the username of a user. Expects a JSON object in the request body. If successful, the JSON object of the updated user info is returned.

DELETE request type

/favorites/:title Deletes a movie from the list of favorites by title. If successful, the updated JSON object of favorite movies is returned.

/users/:name Deletes a user. If successful, the updated JSON object of users is returned.
